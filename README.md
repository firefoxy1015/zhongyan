# 十日终焉：在线桌游

非官方同人项目。目标是将《十日终焉》的正篇、人物番外、游戏规则、角色私密信息与失败分支，转换为可多人协作游玩的原著模式桌游。

## 当前纵切片

- 已完成原文清单锁定与章节索引：`1359` 个正篇节点、`136` 个角色番外、`1` 篇完结感言，共 `1496` 个发布单元。
- 已生成 `content/canon-manifest.json`，记录章节、番外、分组、原文行号与 SHA-256；不把原文全文打包到 Web 端。
- 已完成可游玩的“面试房 / 说谎者”剧本：身份牌翻开、九段讲述、现场推演、秘密投票、成功/失败结算与重开。
- 已完成九人联机房：创建与加入房间、服务端席位、私密房间凭证、阶段同步、桌面讨论、独立投票与九票结算均由 D1 房间状态驱动。
- 已登记视觉证据来源；正式资产只允许来自已登记官方物料或经记录的原文描写，不使用同人图、无来源图片或生成式角色立绘。

## 原文与视觉规则

原文基线文件由本地生成脚本读取，不上传、也不复制进仓库。运行脚本前可用环境变量指定自己的本地副本：

```powershell
$env:CANON_SOURCE = "C:\\path\\to\\十日终焉.txt"
npm run canon:manifest
```

视觉出处与允许用途位于 `content/official-visual-reference.json`：

1. 实体书与官方出版物：标题、封面与卷册语汇。
2. 腾讯视频与番茄动漫官方动画物料：人物、生肖、终焉之地与回响表现。
3. 腾讯视频与柠萌影视官方剧集物料：建筑、服化道与写实材质；不混入默认动画桌游皮肤。
